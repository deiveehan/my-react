Simple youtube react app. 

#Build and deploy

```shell
git clone https://github.com/deiveehan/xplore-react.git
```

## Update Youtube API key
Create a youtube api key and add it in index.js api. 
const API_KEY='<enter-your-api-key>';

## Build and deploy
```shell
npm install
npm start

```

## Run app
http://localhost:8080
